# PrismMTR Assets

## Logo

**Положите файл сюда:** `assets/img/PrismMTRlogo.png`

Этот файл используется как:
- Favicon (иконка вкладки браузера)
- Лого в header/nav на всех страницах
- Лого в модальных окнах (логин и т.д.)

### Рекомендуемые размеры:
- **Минимум:** 64x64 px (для favicon)
- **Рекомендуемо:** 256x256 px или 512x512 px (для четкости на retina)
- **Формат:** PNG с прозрачным фоном

После добавления файла все иконки на сайте автоматически обновятся.
